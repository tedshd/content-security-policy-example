use csp from meta


    <meta http-equiv="Content-Security-Policy" content="script-src 'self';object-src 'none';style-src 'self';child-src 'self';frame-src 'self';">

test in chrome 52, safari 9.1.3, firefox 48.0.2

  • script-src 'self' - only self js
  • object-src 'none' - no object tag
  • style-src 'self' - only self style
  • child-src 'self' - only self iframe
  • frame-src 'self' - only self iframe for safari browser
